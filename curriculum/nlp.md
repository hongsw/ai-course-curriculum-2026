# 자연어정보분석 — 주차별 커리큘럼 (NLP Information Analysis)

**과목 성격**: 한국어 NLP 기초에서 출발해 **LLM 파인튜닝**으로 수렴하는 과목. 토크나이저 벤치마크로 언어 특성을 체감하고, 임베딩·Attention·microGPT 내부를 코드로 뜯어본 뒤, 기말에서 **Unsloth/QLoRA 실제 파인튜닝**을 수행하는, 모델 내부로 파고드는 아크.

## 주차별 설계

| 주차 | 학습 목표 |
|---|---|
| 1 | **문서 포맷 생성형 AI 사례 조사** (CAD·MIDI·SMILES·FHIR 등, "구조 이해 vs 패턴 생성" 논의) |
| 2 | **토크나이저 벤치마크** — 역사 코퍼스로 형태소·BPE·SentencePiece 비교 |
| 3 | **Word2Vec 시멘틱 임베딩** (도메인별 벡터 연산) |
| 4 | **BERT 감성분류 실습** (Colab) |
| 5 | **Self-Attention QKV · 어텐션 시각화** (다의어·대용어 해소) |
| 6 | **microGPT 코드 분석** (Forward Pass · backward · Pre-training) |
| 7 | **다국어 NLP 파이프라인 비교** (중간, 토큰화·임베딩·Attention·모델) |
| 11 | **AI 아키텍처 논문 팀 발표** (Mamba / MoE / VLM / RNN) |
| 12 | 타 팀 발표 청취·정리 |
| 14 | 기말 **주제 선정 문답** (모델·데이터·하이퍼파라미터 계획) |
| 16 | **Unsloth/QLoRA 실제 파인튜닝** (베이스 선택 → 데이터셋 → 학습 → 개선 입증 → export) |

| 아이디어·문제정의 | 20 |
| Unsloth 파인튜닝 활용도 | 20 |
| 데이터셋·하이퍼파라미터 근거 | 25 |
| 베이스 대비 개선도 | 15 |

## 설계 관찰

- 2주차 토크나이저 벤치마크로 **한국어 형태소·서브워드 문제**를 실물로 체감시키는 출발점이 효과적.
- 3→6주차가 임베딩 → Attention → microGPT 내부로 **모델을 아래로 파고드는** 구조 — 기말 파인튜닝의 기반.
- 기말이 이론이 아니라 **직접 파인튜닝한 모델 + 베이스 대비 정량 개선 입증**을 요구. 무료 Colab T4(15GB) 제약 하에서도 전원 재현 가능한 파인튜닝을 수행.
- W14 사전 주제 선정 + 명확한 산출물 규격(repo+보고서+영상) 덕에 제출 누락이 거의 없었다.
